# Code of Conduct

## Our Standards

The Kanboard MCP project is committed to providing a welcoming, inclusive, and respectful environment for everyone — contributors, maintainers, and users alike — regardless of experience level, background, or identity.

We adopt the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as our Code of Conduct. By participating in this project (opening issues, submitting pull requests, joining discussions, or otherwise interacting with the community), you agree to uphold its terms.

The full text of the Contributor Covenant 2.1 is available at:

- HTML: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- Plain text: https://www.contributor-covenant.org/version/2/1/code_of_conduct.txt

## Summary

In short, we expect everyone to:

- **Be respectful** in all interactions — issues, PRs, discussions, code reviews.
- **Welcome newcomers** and help them get started.
- **Give and accept feedback gracefully** — assume good intent, focus on the technical merit.
- **Take responsibility** when mistakes happen, learn from them, and move forward.
- **Prioritize the community's well-being** over individual preferences.

Behavior that the Contributor Covenant identifies as unacceptable will be addressed under the enforcement guidelines below.

## Enforcement

If you experience or witness conduct that violates this Code of Conduct, please report it directly to the project maintainer:

- **Email**: ernesto.coronapaez@gmail.com
- **Subject prefix**: `[conduct] kanboard-mcp:`

All reports will be:

- **Reviewed promptly** (within one week of receipt).
- **Kept confidential** — the reporter's identity is not shared with the reported party without explicit consent.
- **Acted on** following the **Enforcement Guidelines** in the Contributor Covenant 2.1: a graduated response from private clarification, through warnings and temporary bans, up to permanent bans for severe or repeated violations.

The maintainer's enforcement decisions are final for this project. Reporters who disagree with an outcome may seek mediation through the [Contributor Covenant maintainers](https://www.contributor-covenant.org/) directly.

## Scope

This Code of Conduct applies in all project spaces — the GitHub repository (issues, PRs, discussions), the npm package metadata, any official chat or forum channels — and when an individual is officially representing the project in public spaces (conferences, social media posts about the project, etc.).
